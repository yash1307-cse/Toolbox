# Toolbox
Toolbox is a simple desktop app which contain 4 mini apps.
1. Calculator
2. Chant Counter
3. To-Do-List
4. Counter




## Features

- User can do calculations
- User can chant using chant counter
- User can add tasks in To Do List
- User can also count using counter


## Tech Stack


**Language:**  Java

**Database:**  MySQL
